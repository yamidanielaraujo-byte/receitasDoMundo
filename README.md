# 🌍 Receitas do Mundo

Um site dedicado a explorar sabores de diferentes países, com receitas tradicionais e modernas de várias culturas.

## 📌 Sobre o projeto
O **Receitas do Mundo** é uma plataforma onde podes descobrir, aprender e experimentar pratos típicos de todo o mundo, desde entradas até sobremesas.

## 🚀 Funcionalidades
- 🌎 Navegação por países
- 🍝 Lista de receitas com imagens
- 📖 Passo a passo de preparação
- ⭐ Sistema de favoritos
- 🔍 Pesquisa de receitas

## 🛠️ Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Next.js
- TypeScript
